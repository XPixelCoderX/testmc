🎮 Minecraft in Browser

Upload your Minecraft Java Edition JAR file to play directly in your browser using CheerpJ 4.2.

📋 How it works:
1. Upload your Minecraft client JAR file (from .minecraft/versions/<version>/<version>.jar)
2. CheerpJ runs the JAR natively in your browser using WebAssembly
3. The game runs on YOUR computer - no servers or VMs involved
⚠️ Important Notes:
• You must own a legitimate copy of Minecraft to use your JAR file
• You MUST serve this page via an HTTP server (Python: python -m http.server 8080)
• Large JAR files (100MB+) may take time to load
• Minecraft 1.19+ requires Java 17 support (partial in CheerpJ 4.2)
